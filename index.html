Gestor de Actividades

¿De qué se trata el gestor de actividades?

Este proyecto es un Gestor de Actividades que está desarrollado en Python y permite:

  • Registrar usuarios con claves seguras.
  • Iniciar sesión y validar credenciales.
  • Crear, listar y marcar actividades como completadas.
  • Persistencia de datos con SQLite.
